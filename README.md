# Adidas Sales Dashboard 📊

## 📌 Project Overview
This project analyzes **Adidas sales data** to gain insights into:
- Total Revenue and Profit
- Regional Sales Performance
- Product Category Performance
- Sales Channels (Online vs In-Store)

The dashboard provides a clear view of KPIs to help in **business decision-making**.

---

## 📂 Files in this Repository
- `Adidas_Sales_Dashboard.pbix` → Power BI dashboard
- `Adidas_Sales_Data.xlsx` → Dataset used for analysis
- `README.md` → Project documentation

---

## 🚀 How to Use
1. Download the `.pbix` file from this repository.
2. Open it in **Power BI Desktop**.
3. Explore the interactive dashboard and filter sales data by:
   - Product category
   - Region
   - Sales method

## 📊 Key Insights
- **Top Region:** West generates the highest revenue.
- **Most Profitable Category:** Men's Athletic Footwear.
- **Sales Channel:** Online sales are growing faster than in-store.
